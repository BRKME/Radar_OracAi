## 📋 ТЕСТ-КЕЙСЫ

### Edge Cases для проверки:

1. **Низкая волатильность**
   - BB bands сходятся (верхняя ≈ нижняя)
   - Ожидается: bb_position = 50.0

2. **Недостаточно данных**
   - limit=50 для EMA200
   - Ожидается: fallback на current_price

3. **Сетевые проблемы**
   - Временный сбой API
   - Ожидается: 3 retry попытки

4. **Пустые данные**
   - yfinance вернул пустой DataFrame
   - Ожидается: graceful degradation

5. **Длинный прогноз**
   - GPT вернул >4096 символов
   - Ожидается: усечение до 4090
